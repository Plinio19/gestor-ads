# Skill: keyword-research

## Objetivo
Descobrir novas palavras-chave relevantes para expandir alcance qualificado na Linklab.

## Fontes de pesquisa
1. Relatório de termos de busca (Search Terms Report)
2. Google Keyword Planner via API
3. Search Console — queries orgânicas do site lojalinklab.com.br
4. Análise de concorrentes

## Critérios para incluir uma keyword

### ✅ Incluir se:
- Contexto laboratorial claro
- Volume de busca > 10/mês
- CPC estimado dentro do limite por categoria
- Relevante para o portfólio da Linklab

### ❌ Não incluir se:
- Sem contexto laboratorial (ex: "béquer" sem laboratório pode ser culinária)
- Volume < 10 buscas/mês
- CPC estimado > limite máximo da categoria
- Fora do portfólio atual
- Indica uso industrial/doméstico sem contexto lab
