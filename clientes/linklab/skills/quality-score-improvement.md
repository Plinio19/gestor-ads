# Skill: quality-score-improvement

## Objetivo
Melhorar o Quality Score das keywords para pagar menos por clique e ter melhor posição na Linklab.

## Meta: QS >= 7 em todas as keywords prioritárias

## Diagnóstico por QS

### QS 1-4 (Crítico)
- Revisar se keyword é relevante para o grupo de anúncios
- Verificar se o anúncio menciona a keyword
- Checar se a landing page tem o produto específico
- Considerar pausar e recriar em grupo mais específico

### QS 5-6 (Atenção)
- Melhorar copy do anúncio para incluir a keyword
- Verificar correspondência anúncio ↔ landing page
- Adicionar extensões de anúncio

### QS 7-8 (Bom)
- Manter e monitorar
- Testar variações de anúncio para melhorar CTR

### QS 9-10 (Excelente)
- Aumentar lance para capturar mais impressões
- Usar como modelo para outros grupos

## Checklist de relevância para Linklab

### Anúncio deve conter:
- Nome do produto específico do grupo
- Pelo menos um diferencial: entrega rápida, frete grátis SP, NF, 12x sem juros
- CTA claro: "Compre Online", "Veja Preço", "Compre Agora"
- Para reagentes: incluir grau analítico (PA, PA-ACS)

### Landing page deve conter:
- O produto exato que foi buscado
- Preço visível
- Prazo de entrega
- Botão de compra/adicionar ao carrinho claro
- Especificações técnicas do produto
- Informação de frete e parcelamento

## Estrutura ideal de grupo de anúncios
- 1 grupo por produto ou categoria específica
- Keywords do mesmo produto juntas
- Anúncio específico para aquele produto
- Landing page do produto correspondente

Exemplo:
- Grupo "Estufa Laboratorial" → keywords sobre estufa → anúncio sobre estufa → landing page da estufa no lojalinklab.com.br
