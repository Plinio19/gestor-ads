# Skill: quality-score-improvement

## Objetivo
Melhorar o Quality Score das keywords para pagar menos por clique e ter melhor posição.

## O que é Quality Score
Nota de 1-10 do Google baseada em:
- CTR esperado (peso maior)
- Relevância do anúncio
- Experiência na landing page

## Meta: QS >= 7 em todas as keywords prioritárias

## Diagnóstico por QS

### QS 1-4 (Crítico)
- Revisar se keyword é relevante para o grupo de anúncios
- Verificar se o anúncio menciona a keyword
- Checar se a landing page tem o produto específico
- Considerar pausar e recriar em grupo mais específico

### QS 5-6 (Atenção)
- Melhorar copy do anúncio para incluir a keyword
- Verificar correspondência anúncio ↔ landing page
- Adicionar extensões de anúncio

### QS 7-8 (Bom)
- Manter e monitorar
- Testar variações de anúncio para melhorar CTR

### QS 9-10 (Excelente)
- Aumentar lance para capturar mais impressões
- Usar como modelo para outros grupos

## Checklist de relevância para Halogenn

### Anúncio deve conter:
- Nome do produto com grau (ex: "Álcool Etílico PA-ACS")
- Pelo menos uma das: laudo, certificado, entrega rápida, fabricante nacional
- CTA claro: "Solicite Cotação", "Peça pelo WhatsApp"

### Landing page deve conter:
- O produto exato que foi buscado
- Grau analítico visível
- Informação de laudo/certificado
- Forma de contato clara (WhatsApp, formulário)
- Prova social (clientes como Fiocruz, USP)

## Estrutura ideal de grupo de anúncios
- 1 grupo por produto principal
- Keywords do mesmo produto juntas
- Anúncio específico para aquele produto
- Ex: Grupo "Álcool Etílico PA" → keywords sobre álcool PA → anúncio sobre álcool PA → landing page do álcool PA
