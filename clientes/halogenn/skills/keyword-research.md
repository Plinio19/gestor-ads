# Skill: keyword-research

## Objetivo
Descobrir novas palavras-chave relevantes para expandir alcance qualificado.

## Fontes de pesquisa
1. Relatório de termos de busca (Search Terms Report)
2. Google Keyword Planner via API
3. Search Console — queries orgânicas
4. Análise de concorrentes

## Critérios para incluir uma keyword

### ✅ Incluir se:
- Contém grau analítico (PA, PA-ACS, PA-CS, HPLC, analítico)
- Contexto laboratorial claro
- Volume de busca > 10/mês
- CPC estimado dentro do limite
- Relevante para o portfólio da Halogenn

### ❌ Não incluir se:
- Sem especificação de grau ou claramente industrial
- Volume < 10 buscas/mês (muito nichado)
- CPC estimado > limite máximo
- Fora do portfólio atual
