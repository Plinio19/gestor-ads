## Regras de estrutura

### Um grupo de anúncios = um produto/tema
- Nunca misturar álcool com acetona no mesmo grupo
- Keywords do grupo devem ser todas sobre o mesmo produto
- Anúncio deve mencionar o produto específico do grupo

### Match types recomendados
- **Phrase match** `"álcool etílico PA"` — equilíbrio entre alcance e relevância
- **Exact match** `[álcool etílico PA-ACS]` — para termos de alta conversão comprovada
- **Broad match** — EVITAR no início, só usar com lista de negativos robusta

### Nunca fazer
- Broad match sem negativos extensos
- Um grupo para todos os produtos
- Anúncio genérico sem mencionar o produto específico
- Campanha sem extensões configuradas
